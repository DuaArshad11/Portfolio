Projects

Learning Management System (LMS)

A Laravel-based educational platform with role-based access control

Developed a comprehensive Learning Management System with Laravel that implements a complete teacher-student workflow. The system features course management, task distribution, and solution evaluation with distinct interfaces for different user roles.

Database Structure:

  • User Roles: Teachers (pre-defined) and Students (self-registering)
  • Relationships:
    • 1 Teacher → Many Subjects (1:N)
    • Many Students ↔ Many Subjects (N:M)
    • 1 Subject → Many Tasks (1:N)
    • 1 Task → Many Solutions (1:N)

Core Technologies:

  • Laravel 10
  • PHP 8
  • SQLite
  • Bootstrap 5
  • Eloquent ORM

Key Features Implemented:

  • Role-based authentication system
  • CRUD operations for subjects/tasks
  • Solution submission & evaluation
  • Form validation with custom rules
  • Soft delete functionality
  • Responsive admin interfaces

Diary Web Application

A React and Laravel-based personal diary system with a SQLite backend

Developed a dynamic diary web application with a Vite-powered React frontend and Laravel REST API backend. The application allows users to create, edit, and view personal diary entries through an intuitive user interface. The backend uses SQLite for lightweight, local data persistence and is seeded with sample entries for demonstration.

Database Structure:

  • Entries Table: Stores diary entries with fields: ID, Title, Content, Created_At

Core Technologies:

  • React (Vite)
  • Laravel 10
  • PHP 8
  • SQLite
  • REST API
  • Bootstrap 5

Key Features Implemented:

  • Navigation bar with links to Main, Diary, and About pages
  • Static landing page with dynamic entry stats
  • Diary list with titles, dates, and full content display
  • Add and edit diary entries with prefilled forms
  • Seamless routing and page transitions
  • Clean, responsive UI design
